# 发布前检查清单（nas-geek-console）

## 1) 自动化检查

在项目目录执行：

```bash
npm run check:ui
npm run smoke
```

期望：
- `UI contract check passed.`
- `Smoke check passed.`

## 2) 手工页面走查（建议 3~5 分钟）

1. 访问控制台首页，确认出现登录遮罩。
2. 使用控制台密码登录，进入「总览面板」。
3. 点击「刷新概览」，确认健康状态、模型摘要、最近任务能刷新。
4. 进入「Skills 中心」，测试搜索按钮和状态筛选。
5. 进入「记忆工作台」，打开任意文件，确认编辑器可读。
6. 进入「内容工作台」，执行公众号推荐搜索，确认卡片可渲染。

## 3) 发布前确认

- 确认无临时调试代码（console 临时代码、TODO/HACK）
