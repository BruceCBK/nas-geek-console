# V1.1 发布说明：任务中心（Task Center）

## 本次更新

- 新增侧边栏入口「任务中心」。
- 新增任务中心页面，支持：
  - 读取任务列表：`GET /api/tasks`
  - 读取日志列表：`GET /api/logs`
  - 任务筛选：`limit`、`type`、`status`、`target`
  - 任务/日志独立刷新按钮
- 前端沿用现有鉴权请求流程（Bearer Token + `apiJson`）。
- 新增 `npm run verify`，用于一键执行 UI 合约检查 + smoke 测试。

## 手工验证清单

1. 启动服务并登录控制台。
2. 点击侧边栏「任务中心」，确认页面可正常展示。
3. 在「任务筛选」中分别输入 `limit/type/status/target`，点击「应用筛选」，确认任务列表变化。
4. 点击「重置」，确认筛选条件恢复默认并重新加载任务。
5. 点击「刷新任务」，确认任务列表刷新提示与数据更新。
6. 点击「刷新日志」，确认日志列表刷新提示与数据更新。
7. 确认其他页面（总览、Skills、记忆、内容）功能不受影响。
8. 执行以下命令并确认通过：

```bash
node --check public/app.js
npm run check:ui
npm run smoke
```
